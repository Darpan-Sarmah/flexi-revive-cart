Pro add-on for Flexi Revive Cart – adds SMS, WhatsApp, push notifications, dynamic discounts, A/B testing, advanced analytics, and more.

== Description ==

Flexi Revive Cart Pro is the premium extension for the free [Flexi Revive Cart](https://github.com/Darpan-Sarmah/flexi-revive-cart) plugin. It adds powerful features to help you recover more abandoned carts:

**Requires:** Flexi Revive Cart (free version) must be installed and activated first.

= Pro Features =

* **Unlimited Email Reminders** – Up to 10 configurable reminders with Urgency and Incentive/Discount templates
* **Dynamic Discount Coupons** – Auto-generate WooCommerce coupons with configurable type, amount, expiry, and usage limits
* **SMS Reminders** – Send recovery reminders via Twilio or Plivo
* **WhatsApp Messaging** – Individual reminders and bulk campaigns via Twilio or Plivo
* **Push Notifications** – Browser push notifications via OneSignal
* **A/B Subject Line Testing** – Split-test email subjects with automatic winner detection
* **Guest Email Capture Popup** – Capture guest emails before they leave
* **Exit-Intent Popup** – Trigger an offer when users attempt to leave
* **Browse Abandonment** – Track product views and send follow-up emails
* **Advanced Analytics** – Recovery analytics by channel (Email, SMS, WhatsApp, Push)
* **CSV Export** – Export abandoned/recovered cart data
* **REST API** – Full API for headless/mobile integrations

= How It Works =

1. Install and activate the free Flexi Revive Cart plugin
2. Install and activate Flexi Revive Cart Pro
3. Enter your license key in **Flexi Revive Cart → Pro License**
4. Configure Pro features in Settings tabs (Discounts, SMS, WhatsApp, Push, Popups, etc.)

= License =

A valid license key is required to activate Pro features. License keys are tied to your domain and include an expiration date.

== Installation ==

1. Ensure Flexi Revive Cart (free version) is installed and activated
2. Upload the `flexi-revive-cart-pro` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu
4. Go to Flexi Revive Cart → Pro License and enter your license key
5. Configure Pro features in Settings
